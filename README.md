# farm.lua

A [MacroQuest](https://www.macroquest.org/) Lua script for the [Project Lazarus](https://www.lazaruseq.com/) EverQuest emulator server. Automates NPC farming with configurable target lists, partial name matching, combat, and corpse looting.

## Usage

```
/lua run farm
```

## Configuration

Edit the `config` table at the top of `farm.lua` to set target NPC names, search radius, rest HP/mana thresholds, loot delay, and class abilities.
